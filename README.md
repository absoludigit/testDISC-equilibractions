L’application DISC EquilibrActions propose un test interactif permettant d’identifier votre profil comportemental via le test DISC. Ce langage psychologique a été dévéloppé par William Marston, juriste et psychologue, qui est le principal initiateur du DISC.

Ce modèle distingue quatre styles clés :

D : Dominant (Rouge) – Axé sur l’action, les résultats, le leadership

I : Influent (Jaune) – Axé sur la communication, l’enthousiasme, la motivation

S : Stable (Vert) – Axé sur la coopération, la sécurité, le rythme régulier

C : Conforme (Bleu) – Axé sur la rigueur, l’analyse, la conformité aux règles

L'outil vise à mieux se connaître et améliorer ses relations et interactions en contexte professionnel ou personnel, sans jamais enfermer l'utilisateur dans une case.

Fonctionnalités
Test en 15 questions, chacune clairement alignée sur les axes DISC (Introversion/Extraversion – Tâche/Relation)

Interface entièrement en français, claire et accessible

Affichage pédagogique : explication du modèle DISC avant le test

Rapport individuel structuré en 7 parties :

Profil principal et secondaire

Positionnement sur les axes DISC (cartographie théorique)

Analyse comportementale détaillée

Forces et axes d’amélioration

Suggestions pour interagir efficacement avec son profil

Conseils management/collaboration

Domaines professionnels recommandés

Utilisation
Ouvrez l’application dans votre navigateur.

Lisez l’introduction sur le modèle DISC.

Cliquez sur “Commencer le test”.

Répondez aux 15 questions à choix unique.

À la fin, consultez votre rapport personnalisé DISC : axes, forces, faiblesses, conseils pratiques.

(Facultatif) Recommencez le test ou partagez vos résultats.
